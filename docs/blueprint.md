# **App Name**: PostPipe Pro

## Core Features:

- Centralized Authentication: Firebase authentication system with shared session management via secure cookies across all subdomains (postpipe.in, forms.studio.postpipe.in, workflows.studio.postpipe.in).
- Static Form Builder: Visual form builder accessible at forms.studio.postpipe.in to create static forms with field validation and HTML/JS embed code generation.
- Form Submissions Overview: Read-only display of form submissions within the static forms studio (forms.studio.postpipe.in).
- Dynamic Workflow Template Marketplace: Template marketplace hosted at workflows.studio.postpipe.in, offering template cards with tags, filters, CLI copy, AI prompt copy, and the ability to save templates to user profiles.
- Agentic AI Prompt Tool: Incorporate pre-formatted AI prompts within the workflow templates, with instructions for users to generate frontends, without generating backend. One-click copy functionality is available.
- Marketing Landing Page: Landing page on postpipe.in with hero section, feature highlights, call-to-action buttons, documentation access, and pricing information.
- Authentication Middleware: Middleware to verify user authentication and redirect unauthenticated users to the login page (postpipe.in/login) across all subdomains.

## Style Guidelines:

- Primary color: Dark vibrant purple (#9D4EDD) to evoke creativity and technical prowess.
- Background color: Dark desaturated purple (#211A2C) for a dark-first design.
- Accent color: Bright pink (#E5388B) for interactive elements and call-to-action buttons to draw attention.
- Body font: 'Inter' (sans-serif) for a modern and neutral look.
- Headline font: 'Space Grotesk' (sans-serif) for a computerized, techy feel.
- Code font: 'Source Code Pro' (monospace) for code snippets.
- Use a set of minimalistic icons appropriate for a developer-centric interface.
- Implement a responsive layout optimized for various screen sizes to ensure accessibility and usability.
- Subtle transitions and animations for UI elements to improve user experience without being intrusive. Use Lenis smooth scroll.
