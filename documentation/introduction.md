PostPipe offers two main ways to integrate your data: **Static** and **Dynamic**.

## 1. Static (The Connector)

Use this if you simply want to connect an existing database to the PostPipe SaaS.

- **Goal**: Securely Post Data Payloads to your configured Database.
- **Tool**: The **PostPipe Connector**.
- **Guide**: [Setup Static Connector](/docs/guides/static-connector)

## 2. Dynamic (CLI Components)

Use this if you want to build NextJS Backends from scratch using our pre-built modules and CLI system.

- **Goal**: Scaffold Authentication, E-commerce, or Shop systems.
- **Tool**: The **PostPipe CLI Ecosystem**.
- **Guide**: [Use CLI Components](/docs/guides/cli-components)
